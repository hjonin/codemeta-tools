# codemeta-tools

A set of tools to support the creation of metadata files (including but not limited to `codemeta.json` files) for software.

## Installation

**Node.js (>= 18) + npm**

```bash
npm install codemeta-tools
```

## Run locally

Install dependencies:
```bash
npm install
```

Run tests:
```bash
npm run test 
```

Publish:
```bash
npm version [version]
npm publish
```
