# codemeta-tools

A set of tools to support the creation of metadata files (including but not limited to `codemeta.json` files) for software.

## Run locally

Install dependencies:
```bash
npm install
```

Run validator on test data:
```bash
ts-node index.ts
```

Run tests:
```bash
npm run test 
```
